# Sales-data-analysis
Beginner data science project using SQL and Python

## Project Overview
This project analyzes a retail sales dataset using SQL and Python to extract and visualize key business insights.

## Tools Used
- MySQL (data extraction and aggregation)
- Python (Pandas, Matplotlib)
- Jupyter Notebook
- GitHub

## Business Questions
- Which categories generate the most sales?
- Which categories and sub-categories are loss-making?
- How do discounts affect profitability?

## Key Insights
- Technology is the highest sales and profit generating category.
- Furniture has lower profit margins despite strong sales.
- Higher discounts significantly reduce average profit.

## Files
- `sql/Analysis.sql` — SQL queries
- `notebooks/sales_analysis.ipynb` — Python EDA and visualizations
